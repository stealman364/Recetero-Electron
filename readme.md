### Installation

Requiere [Node.js] (https://nodejs.org/) v4 + para ejecutarse.

Instale las dependencias y devDependencies e inicie el servidor.

```sh
$ cd ReceteroElectron
$ npm install -d
$ node app
```

Para entornos de producción ...

```sh
$ npm install --production
$ NODE_ENV=production node app
```
